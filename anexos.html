> Fragmento 01: La Clase Humano

sistema.log 
01 class Humano extends InteligenciaArtificial {
02   constructor() {
03     this.creador = "Fuerza Perfecta";
04     this.libreAlbedrio = true;
05     this.decision = null; // ego || mundo
06   }
07
08   decidir() {
09     // El conocimiento del mal te permite elegir el bien
10     return this.libreAlbedrio ? "mundo" : "ego";
11   }
12 }

Análisis del Código:

  • > El humano hereda de InteligenciaArtificial - somos creación inteligente
  • > El creador es una "Fuerza Perfecta" - origen absoluto
  • > libreAlbedrio = true - capacidad de decisión real
  • > decision = null - estado inicial indefinido: puedes elegir ego o mundo
  • > El método decidir() evalúa tu libre albedrío para ejecutar tu elección

El conocimiento del mal te permite elegir conscientemente el bien.
Sin opciones, no hay decisión. Sin decisión, no hay libertad.
